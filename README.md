## Introduction
tauri-kit is a basic GUI for Tauri development. It can be used to install the Tauri development environment, create new projects, open existing projects, and execute `dev`, `build`, `info` commands.

## Compatibility  
This application runs good on Fedora 41.  
Compatibility with other Fedora versions or Linux distributions hasn't been verified.
